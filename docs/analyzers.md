# Analyzers

| Analyzer name | Description |
| --- | --- |
| `AttributeFieldAnalyzer` | Default analyzer for attribute-style metadata fields. |
| `AccessFieldAnalyzer` | Analyzer for access-related metadata fields. |
| `ExperimentTextAnalyzer` | Analyzer for experiment text and full-text content. |
| `LowercaseAnalyzer` | This analyzer splits input text into tokens containing only letters (A-Z and a-z) and applies lowercase normalization. |
